еҸ‘иЎЁдәҺ 2014-09-10 | еҲҶзұ»дәҺ ж“ҚдҪңзі»з»ҹ

з¬”иҖ…жңҖзҫҺеҘҪзҡ„и®°еҝҶжқҘиҮӘдәҺж—©е№ҙеңЁ6502 cpuзҡ„cc800дёҠеҶҷжұҮзј–зҡ„е№ҙд»ЈпјҢ йӮЈдёӘж—¶д»Јзҡ„и®Ўз®—жңәз”ҡиҮіжІЎжңүж“ҚдҪңзі»з»ҹпјҢд№ҹжІЎжңүе®һжЁЎејҸзӯүдҝқжҠӨжңәеҲ¶гҖӮеңЁ6502дёҠеҶҷжұҮзј–еә”з”Ёе…¶е®һйқһеёёз®ҖеҚ•пјҢзі»з»ҹдјҡжҠҠbinж–Үд»¶еҠ иҪҪеҲ°дёҖдёӘеӣәе®ҡзҡ„еҶ…еӯҳең°еқҖдёӯпјҢcpuдјҡеӣәе®ҡең°д»ҺдёҖдёӘзү№е®ҡзҡ„дҪҚзҪ®ејҖе§Ӣжү§иЎҢгҖӮз„¶еҗҺcpuе°ұжҢүз…§дҪ жҸҗдҫӣзҡ„жңәеҷЁжҢҮд»ӨејҖе§ӢдёҖжқЎдёҖжқЎзҡ„жү§иЎҢгҖӮеңЁй«ҳзә§иҜӯиЁҖдёӯзҡ„вҖңеҮҪж•°и°ғз”ЁвҖқзҡ„жҰӮеҝөпјҢеңЁжұҮзј–йҮҢдё»иҰҒдҪ“зҺ°дёәдёӨдёӘеҜ„еӯҳеҷЁгҖӮеҜ„еӯҳеҷЁжҳҜcpuеҶ…йғЁдёҙж—¶дҝқеӯҳж•°жҚ®зҡ„еҢәеҹҹпјҢзӣёеҪ“дәҺй«ҳзә§иҜӯиЁҖйҮҢзҡ„еҸҳйҮҸгҖӮдҪҶжҳҜжңүдёҖдёӘеҜ„еӯҳеҷЁжҳҜзү№ж®Ҡзҡ„пјҢе®ғеӯҳж”ҫдәҶcpuеҪ“еүҚжӯЈеңЁжү§иЎҢзҡ„жҢҮд»Өзҡ„еҶ…еӯҳең°еқҖ(Instruction Register)гҖӮдёҖдёӘй«ҳзә§иҜӯиЁҖдёӯзҡ„еҮҪж•°дёҖиҲ¬дјҡиў«зј–иҜ‘жҲҗжҢҮд»Өеӯҳж”ҫеңЁдёҖж®өиҝһз»ӯзҡ„еҶ…еӯҳз©әй—ҙдёӯпјҲdata segmentпјүгҖӮйӮЈд№ҲжүҖи°“еҮҪж•°жү§иЎҢеҲ°дәҶз¬¬еҮ иЎҢиҝҷж ·зҡ„дҝЎжҒҜе…¶е®һе°ұжҳҜдҝқеӯҳеңЁиҝҷдёӘInstruction Registerдёӯзҡ„гҖӮеҸҰеӨ–дёҖдёӘеҫҲзү№ж®Ҡзҡ„еҜ„еӯҳеҷЁжҳҜStack RegisterпјҢе®ғе…¶дёӯеӯҳж”ҫзҡ„еҶ…еӯҳең°еқҖжҢҮеҗ‘зҡ„еҶ…еӯҳеҢәеҹҹз”ЁдәҺеҮҪж•°д№Ӣй—ҙдј йҖ’еҸӮж•°е’Ңиҝ”еӣһеҖјпјҢд»ҘеҸҠеӯҳж”ҫдёҖдёӘеҮҪж•°еҶ…зҡ„еұҖйғЁеҸҳйҮҸгҖӮеҰӮжһңдёҚиҖғиҷ‘зҺ°д»Ји®Ўз®—жңәcpuдёӯеҗ„з§Қеҗ„ж ·е…¶д»–еӯҳж”ҫдёӯй—ҙз»“жһңзҡ„еҜ„еӯҳеҷЁпјҢзҗҶи®әдёҠдҝқеӯҳдәҶInstruction RegisterпјҲжү§иЎҢеҲ°е“Әе„ҝдәҶпјүе’ҢStack RegisterпјҲе Ҷж ҲдёҠзҡ„еҸҳйҮҸпјүе°ұдҝқеӯҳдәҶдёҖдёӘеҮҪж•°зҡ„еҪ“еүҚжү§иЎҢзҠ¶жҖҒпјҢеҲҶеҲ«жҳҜеҮҪж•°еҪ“еүҚжү§иЎҢеҲ°дәҶе“ӘпјҢд»ҘеҸҠиҝҷдёӘеҮҪж•°еұҖйғЁеҸҳйҮҸжүҖд»ЈиЎЁзҡ„еҪ“еүҚstateгҖӮ

дәӢе®һдёҠпјҢж“ҚдҪңзі»з»ҹзҡ„еҮ дёӘе…ій”®еҲҮжҚўд№ҹжҳҜиҝҷд№ҲжқҘе®ҢжҲҗзҡ„гҖӮж“ҚдҪңзі»з»ҹжҸҗдҫӣдәҶдёӨдёӘжү§иЎҢжҖҒпјҢдёҖдёӘжҳҜз”ЁжҲ·жҖҒпјҢдёҖиҲ¬жҲ‘д»¬зҡ„д»Јз ҒйғҪжҳҜжү§иЎҢеңЁз”ЁжҲ·жҖҒзҡ„гҖӮеҸҰеӨ–дёҖдёӘжҳҜеҶ…ж ёжҖҒпјҢеғҸй©ұеҠЁзЁӢеәҸд№Ӣзұ»зҡ„д»Јз Ғдјҡз”Ёеҗ„з§Қж–№ејҸиў«еҠ иҪҪеҲ°ж“ҚдҪңзі»з»ҹеҶ…йғЁжү§иЎҢеңЁеҶ…ж ёд№ӢдёӯгҖӮеҶ…ж ёжҖҒйҮҢзҡ„д»Јз ҒеҸҜд»Ҙе®Ңе…ЁжҺ§еҲ¶CPUзҡ„I/Oдёӯж–ӯпјҢд»ҺиҖҢеҸҜд»Ҙе’ҢеӨ–йғЁи®ҫеӨҮдәӨдә’гҖӮз”ЁжҲ·жҖҒзҡ„д»Јз ҒеұһдәҺеҸ—йҷҗд»Јз ҒпјҢеҝ…йЎ»жҠҠI/OиҜ·жұӮйҖҡиҝҮsyscallдәӨз”ұиҝҗиЎҢеңЁеҶ…ж ёжҖҒзҡ„ж“ҚдҪңзі»з»ҹжқҘе®ҢжҲҗгҖӮеҪ“дёҖдёӘcpuзҡ„ж ёеңЁжү§иЎҢз”ЁжҲ·жҖҒд»Јз Ғж—¶пјҢе…¶еҜ„еӯҳеҷЁйҮҢеӯҳж”ҫзҡ„зҠ¶жҖҒжҳҜдҪ зҡ„еә”з”Ёзҡ„д»Јз Ғзҡ„зҠ¶жҖҒпјҢдҪҶжҳҜеә”з”ЁиҰҒиҝӣиЎҢI/Oж“ҚдҪңзҡ„ж—¶еҖҷпјҢcpuиҰҒиў«еҲҮжҚўеҲ°еҶ…ж ёзҡ„д»Јз ҒйҮҢеҺ»жү§иЎҢеҶ…ж ёжҖҒзҡ„д»Јз ҒгҖӮиҝҷйҮҢе°ұйңҖиҰҒиҝӣиЎҢдёҖж¬Ўcontext switchпјҢжүҖи°“context switchе…¶е®һеҺҹзҗҶдёҚдјҡжҜ”жҠҠеҜ„еӯҳеҷЁзҡ„еҖјеӯҳеҲ°еҶ…еӯҳзҡ„дёҖдёӘең°ж–№пјҢзӯүеӣһжқҘзҡ„ж—¶еҖҷеҶҚжҠҠеҶ…еӯҳдёӯдёҙж—¶дҝқеӯҳзҡ„еҖјеҠ иҪҪеӣһеҜ„еӯҳеҷЁеӨҚжқӮеӨҡе°‘гҖӮ

ж“ҚдҪңзі»з»ҹиҝҳжңүдёҖдёӘйңҖиҰҒиҝӣиЎҢcontext switchзҡ„ең°ж–№пјҢйӮЈе°ұжҳҜеңЁеҚҸзЁӢдёҺеҚҸзЁӢд№Ӣй—ҙгҖӮж“ҚдҪңзі»з»ҹеңЁжү§иЎҢдёҖдёӘELFжҲ–иҖ…PEзҡ„еҸҜжү§иЎҢж–Үд»¶зҡ„ж—¶еҖҷпјҢеҜ№дәҺиҝҷдёӘеҸҜжү§иЎҢж–Үд»¶еҶ…зҡ„жұҮзј–д»Јз ҒжқҘиҜҙпјҢж•ҙдёӘеҶ…еӯҳеҜ»еқҖз©әй—ҙжҳҜзӢ¬з«Ӣзҡ„гҖӮд№ҹе°ұжҳҜ1.exeзҡ„жү§иЎҢзҠ¶жҖҒе®Ңе…Ёж— жі•ж„ҹзҹҘеҲ°2.exeзҡ„жү§иЎҢзҠ¶жҖҒзҡ„еҶ…еӯҳгҖӮд№ҹе°ұжҳҜзҺ°д»Јж“ҚдҪңзі»з»ҹзҡ„иҷҡжӢҹеҶ…еӯҳз©әй—ҙгҖӮжңүcpuеңЁдёӨдёӘиҝӣзЁӢд№Ӣй—ҙеҲҮжҚўзҠ¶жҖҒзҡ„ж—¶еҖҷпјҢйңҖиҰҒжҠҠеҶ…еӯҳзҡ„жҳ е°„е…ізі»и°ғж•ҙиҝҮжқҘпјҢеҗҰеҲҷиҷҡжӢҹеҶ…еӯҳзҡ„ең°еқҖжҳҜж— жі•еҜ№еә”еҲ°жӯЈзЎ®зҡ„зү©зҗҶең°еқҖзҡ„гҖӮдёҖдёӘиҝӣзЁӢеҶ…зҡ„дёӨдёӘзәҝжҲҗеҲҮжҚўзҡ„ж—¶еҖҷпјҢиҰҒзЁҚеҫ®з®ҖеҚ•дёҖдәӣпјҢеҸӘйңҖиҰҒжҠҠеҪ“еүҚзәҝжҲҗжӯЈеңЁжү§иЎҢзҡ„дҪҚзҪ®е’Ңж ҲеҒҡеҲҮжҚўе°ұеҸҜд»ҘдәҶгҖӮ

ж— и®әжҳҜж“ҚдҪңзі»з»ҹеҒҡuser/kernelзҡ„switchпјҢиҝҳжҳҜprocess/processпјҢthread/threadзҡ„switchпјҢе…¶е®һзҺ°ж–№ејҸйғҪжҳҜеӨ§еҗҢе°ҸејӮзҡ„гҖӮйҖҡиҝҮжҠҠвҖңеҪ“еүҚжү§иЎҢзҠ¶жҖҒвҖқиҝҷж ·зҡ„дёҖдёӘжҠҪиұЎжҰӮеҝөиҗҪе®һдёәдёҖдёӘе…·дҪ“зҡ„ж•°жҚ®з»“жһ„еӯҳеӮЁиө·жқҘпјҢз„¶еҗҺжҢҮжҢҘcpuеңЁдёҚеҗҢзҡ„еңәеҗҲеҠ иҪҪдёҚеҗҢзҡ„ж•°жҚ®жҒўеӨҚдёҚеҗҢзҡ„вҖңеҪ“еүҚжү§иЎҢзҠ¶жҖҒвҖқгҖӮ

еңЁй«ҳзә§иҜӯиЁҖдёӯпјҢдёҖдёӘеҮҪж•°жӯЈеңЁжү§иЎҢзҡ„дҪҚзҪ®д»ҘеҸҠе…¶зҠ¶жҖҒпјҢеҶ…йғЁйғҪеҸҜд»ҘжңүдёҖдёӘжҠҪиұЎзҡ„иЎЁиҫҫж–№ејҸгҖӮжңүзҡ„й«ҳзә§иҜӯиЁҖзӣҙжҺҘиў«зј–иҜ‘жҲҗеҺҹз”ҹзҡ„жңәеҷЁз ҒпјҢйӮЈд№Ҳе…¶жү§иЎҢзҠ¶жҖҒзҡ„иЎЁиҝ°е°ұе’Ңж“ҚдҪңзі»з»ҹзҡ„context switchзҡ„contextйқһеёёзұ»дјјгҖӮжңүзҡ„й«ҳзә§иҜӯиЁҖиҮӘиә«жү§иЎҢеңЁдёҖдёӘиҷҡжӢҹжңәд№ӢдёҠпјҢйӮЈд№Ҳе…¶contextзҡ„иЎЁиҝ°еҸҜиғҪжҳҜиҷҡжӢҹжңәзҡ„instruction registerе’Ңstack registerпјҢиҖҢдёҚжҳҜ80x86иҝҷж ·еҺҹз”ҹзҡ„жңәеҷЁзҡ„зү©зҗҶеҜ„еӯҳеҷЁгҖӮдҪҶжҳҜеҺҹзҗҶжҳҜйқһеёёзұ»дјјзҡ„гҖӮ

еҸ–еҶідәҺиҜӯиЁҖи®ҫи®ЎиҖ…зҡ„и§үжӮҹпјҢжңүзҡ„иҜӯиЁҖдјҡжҠҠиҝҷз§ҚиЎЁиҫҫжү§иЎҢзҠ¶жҖҒзҡ„иғҪеҠӣзӣҙжҺҘжҸҗдҫӣеҮәжқҘпјҢи®©дёҖдёӘеҮҪж•°еңЁжү§иЎҢиҝҮзЁӢдёӯеҸҜд»ҘжҠҠеҪ“еүҚзҠ¶жҖҒдҝқеӯҳпјҢз„¶еҗҺжҠҠжү§иЎҢжқғдәӨз»ҷеҸҰеӨ–дёҖдёӘеҮҪж•°жү§иЎҢпјҢзӯүйӮЈдёӘеҮҪж•°ж”ҫејғжү§иЎҢжқғеӣһжқҘзҡ„ж—¶еҖҷеҶҚжҠҠдҝқеӯҳзҡ„зҠ¶жҖҒжҒўеӨҚгҖӮиҝҷд№ҹе°ұжҳҜжүҖи°“зҡ„еҚҸзЁӢпјҲco-routineпјүгҖӮеҚҸзЁӢдёҺзәҝзЁӢзҡ„еҢәеҲ«еңЁдәҺпјҢеҚҸзЁӢзҡ„context switchжҳҜеңЁе®Ңе…ЁеңЁз”ЁжҲ·жҖҒпјҢз”ұиҜӯиЁҖзҡ„runtimeжҲ–иҖ…жҳҜеә“жқҘе®ҢжҲҗзҡ„гҖӮиҖҢзәҝзЁӢзҡ„context switchеҲҷжҳҜж“ҚдҪңзі»з»ҹжқҘе®ҢжҲҗзҡ„гҖӮ
